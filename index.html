Profolio

Under Construction

Under Construction is a site for all DIY lovers. Users can create there own DIY projects or post a link to an existing one which will return information based on the link. In the spirit of DIY this site was completely designed from the ground up without the aid of Bootstrap or Foundation.

Technology Stack

Rails, Postgres, Ajaxs, jQuery

Gin Rummy

A remake of the classic card game Gin Rummy. Two players race to see who can discard their hands first. On this project I uses SASS to render the cards and create fluid animation. We also wrote Mocha tests for all the function which greatly reduced our time spent debugging.

Technology Stack

Javascript, Handlebars JS, SASS, Websockets IO, ExpressJS, Mocha Test

Jungle

Jungle e-commerce is an example of how I picked up an unfamiliar language and run with it. In Jungle customers are able to make purchases and leave reviews about the product. An email will be sent out with every order.

Technology Stack

Ruby on Rails, Sass, Stripe, Postgres, Bootstrap, Rspec

Chatty App

A live chat application where anyone can join and chat with one another. An example of learning a new framework and building on it. Users can chat in real time and there's a display that shows all online users.

Technology Stack

React, Javascript, Websocket, SASS

Tweeter

Tweeter is a single-page Twitter clone. Users are able to post messages of up to 140 characters, and browse through the list of posts from different users. This app I incorporated HTML, CSS, JS, jQuery and AJAX on the front-end, and Node, Express and MongoDB on the back-end.

Technology Stack

ExpressJS, CSS, JS, jQuery, MongoDB, AJAX
